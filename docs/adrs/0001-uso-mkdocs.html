ADR-001: Uso de MkDocs para Documentación

Estado: Aceptado

Fecha: 2025-11-20

Contexto

El proyecto necesita una forma centralizada y accesible de mantener la documentación técnica y funcional. Los documentos Markdown dispersos son difíciles de navegar para usuarios no técnicos.

Decisión

Se eligió MkDocs con el tema Material for MkDocs para centralizar la información y permitir una navegación rápida.

Consecuencias

  • Positivas: Centralización, búsqueda rápida, separación de contenido y presentación.
  • Negativas: Requiere un paso de compilación.
