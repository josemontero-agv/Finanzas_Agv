ADR-002: Plataforma Externa para Reportería

Estado: Aceptado

Fecha: 2025-11-14

Contexto

Odoo presenta lentitud en la generación de reportes complejos y personalizados de cobranzas, lo que afecta la operatividad del equipo financiero.

Decisión

Desarrollar una plataforma externa (Finanzas AGV) que consuma datos de Odoo vía API y los procese de forma optimizada para la visualización y exportación.

Consecuencias

  • Positivas: Mayor velocidad, UX personalizada, menor carga en Odoo.
  • Negativas: Necesidad de mantener un sistema adicional y asegurar la sincronización de datos.
