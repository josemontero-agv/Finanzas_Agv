Manual de Usuario y Guía Funcional

1. Introducción

Finanzas AGV es una plataforma de reporting y gestión que consume datos directamente de Odoo ERP para optimizar los procesos de Créditos y Cobranzas.

Objetivo: Facilitar la trazabilidad entre Facturas y Letras, permitiendo un control preciso de los documentos pendientes de firma.

2. Gestión de Letras por Firmar

El módulo principal permite visualizar las letras que aún no han sido aceptadas por los clientes.

Funcionalidades Clave:

  • Filtros Avanzados: Búsqueda por Cliente, RUC, Número de Letra y Estado (Vigente / Por Recuperar).
  • Previsualización: Permite ver el diseño final del correo antes de enviarlo.
  • Prevención de Duplicados: El sistema alerta si se intenta reenviar una letra en la misma sesión.

3. Envío de Correos

Los correos se envían de forma consolidada. Si un cliente tiene varias letras, recibirá un único correo profesional con el detalle completo.

Pasos para el envío:

  1. Filtrar y seleccionar las letras deseadas.
  2. Hacer clic en "Previsualizar" para validar los datos.
  3. Hacer clic en "Enviar Correos".
  4. Confirmar en el modal de validación.

4. Integración con Odoo (Cuenta 42)

El sistema utiliza la relación a través de account.bill.form (Planillas) para vincular de forma sólida las Facturas originales con las Letras generadas.

Campo Origen Odoo Descripción
Factura account.move (name) Documento de venta original.
Letra account.move (ref) Documento de crédito aceptado.
Planilla account.bill.form Vínculo entre factura y letra.
