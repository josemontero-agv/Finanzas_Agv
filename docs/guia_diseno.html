Guía de Diseño y Paleta de Colores

Este documento resume los estándares visuales utilizados en la documentación del proyecto Finanzas AGV.

1. Concepto de Diseño

El diseño está orientado a un entorno Corporativo y Profesional, priorizando la legibilidad técnica y una presentación impecable.

  • Layout: Estructura de dos columnas con barra lateral (TOC) fija.
  • Tipografía: Poppins para texto general y Fira Code para bloques técnicos.

2. Paleta de Colores

Elemento Color Hex Uso
Primario #714B67 Títulos y botones.
Secundario #5A3C52 Hover y énfasis.
Acento #875A7B Bordes y badges.

3. Bloques de Información
